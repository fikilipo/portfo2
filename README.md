# Эхо Рунета

Иммерсивный одностраничник, который **физически перестраивает** свой UI, типографику и поведение в зависимости от выбранной эпохи Рунета:

- **/01 — Дикий Запад (1997 — 2003):** табличная вёрстка, ICQ, гифки, гостевая книга, кастомный курсор.
- **/02 — Золотой Век (2007 — 2012):** скевоморфизм, ВК-стена, демотиватор-генератор, «Дуров, верни стену».
- **/03 — Экосистема (2020 — 2026):** glassmorphism, Three.js, AI-консоль с командой `/glitch`.

Между эпохами — **глитч-разрыв реальности** (RGB-сдвиг + scanlines + flash + камера-shake).

ТЗ — в файле [TZ.md](./TZ.md). Исходный бриф — в файле [TZ](./TZ).

## Стек

- Vite 5 + React 18 + TypeScript
- Tailwind CSS
- GSAP (timeline для глитч-перехода)
- Three.js + `@react-three/fiber` + `drei` (только эпоха 3, lazy-loaded)
- Web Audio API для синтезированных SFX (никаких бинарных ассетов)

## Скрипты

```bash
npm install
npm run dev          # локальный dev-сервер
npm run build        # production-сборка в dist/
npm run preview      # локальный просмотр production-билда
npm run typecheck
npm run lint
```

## Особенности

- Deep-link: `?era=1|2|3` сохраняется в URL, работает «назад» в браузере.
- `prefers-reduced-motion` уважается — глитч заменяется на короткий fade.
- Все звуки выключены по умолчанию, кнопка mute — в правом нижнем углу.
- Никаких пользовательских данных не отправляется наружу: гостевая книга и стена — только в `localStorage`.
- Three.js-сцена — отдельный чанк, грузится только при входе в эпоху 3.

## Пасхалки
