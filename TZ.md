# ТЗ — «Эхо Рунета» (Echo of Runet)

> Иммерсивный веб-проект — цифровое исследование души российского интернета, который **физически перестраивает свою структуру, логику и эстетику** в зависимости от выбранной временной эпохи.

Этот документ — улучшенная и детализированная версия исходного ТЗ. Сохранены все идеи и философия оригинала, добавлены: структура страниц, конкретные блоки контента, требования к UX/перформансу/доступности, набор «артефактов» эпох, технические рамки и критерии приёмки.

---

## 1. Миссия и философия

- Сайт — **перформанс**, а не просто переключатель тем.
- Каждая эпоха — **отдельный микро-мир** со своим визуальным языком, типографикой, звуком, поведением курсора и UI-микро-взаимодействиями.
- Между эпохами — **глитч-разрыв реальности**, а не переход.
- Контент — на живом, аутентичном русском языке. Кириллица — главный герой типографики.

---

## 2. Структура сайта (одностраничник)

1. **Boot-screen / интро** — короткая загрузка в духе CRT/модема (≤ 2 c) с возможностью пропуска.
2. **Era Switcher** — фиксированный таймлайн-переключатель эпох (вверху или сбоку), всегда виден, с подсказкой текущей эпохи.
3. **Главная сцена эпохи** — Hero: гигантский кириллический заголовок + манифест эпохи + декоративные «артефакты».
4. **Манифест эпохи** — поэтичный текст 2–3 абзаца, передающий настроение времени.
5. **Артефакты эпохи** — 3–5 интерактивных элементов, характерных для эпохи (см. §4).
6. **Хроника / факты** — 4–6 культовых событий/мемов/сайтов того времени, оформленные в стилистике эпохи.
7. **Гостевая книга** (только эра 1) / **Стена** (только эра 2) / **AI-консоль** (только эра 3) — интерактивный блок, который пользователь может «потрогать».
8. **Подвал** — счётчик посещений / копирайт / ссылка-пасхалка.

При переключении эпохи **глитч-перекадровка**: RGB-сдвиг + scanlines + смена палитры/типографики/звука.

## 3. Эпохи

### 3.1. «Дикий Запад» (1997–2003)
- **Дух:** «Народ.ру», fido, форумы, чаты «Кроватка», ICQ, под construction-гифки.
- **Палитра:** серый системный (#C0C0C0), кислотные ссылки (#0000EE / #800080), жёлтый, красный, чёрный.
- **Типографика:** Times New Roman / Courier New / Comic Sans, выравнивание `<center>`, заглавные **Ж**ИРНЫЕ.
- **Лэйаут:** **табличная вёрстка**, рамки 3D, хаотичные отступы, фоновая тайл-текстура.
- **Артефакты:**
  - Кастомный курсор со звёздочками-следом.
  - `<marquee>`-бегущая строка.
  - Анимированные «огоньки», «under construction», вращающаяся почта-конверт.
  - Счётчик посещений «Вы — посетитель №…».
  - Гостевая книга (форма имя+сообщение, локально хранит записи).
  - Hit-counter c семисегментным шрифтом.
- **Звук:** одноразовый «диал-ап» при входе в эпоху (с глобальным mute-тогглом).

### 3.2. «Золотой Век / Дуров, верни стену» (2007–2012)
- **Дух:** Web 2.0, ранний ВКонтакте, Хабр, Лепра, демотиваторы, ЖЖ, скевоморфизм.
- **Палитра:** ВК-синий (#45688E / #5B7FA6), белый, мягкие градиенты, глянец.
- **Типографика:** Tahoma / Verdana / Arial — компактные плотные интерфейсы.
- **Лэйаут:** двух/трёхколоночный, «карточки», скруглённые углы, тонкие тени, glossy-кнопки.
- **Артефакты:**
  - **«Стена»** — посты с лайками/комментами; пользователь может оставить пост.
  - Демотиватор-генератор (загрузил картинку → подпись → чёрная рамка).
  - Кнопка «Дуров, верни стену» с пасхалкой.
  - Всплывающее уведомление в стиле ICQ/QIP.
  - Профиль-карточка с «онлайн»-точкой.
- **Звук:** «дзынь» уведомления при действии.

### 3.3. «Экосистема / Нейро-Синтез» (2020–2026)
- **Дух:** Яндекс/Сбер/Тинькофф/VK Tech, AI-ассистенты, минимализм, темнота.
- **Палитра:** глубокий чёрный/графит, акценты — электрик-фиолет, кислотный лайм, соляризованные градиенты.
- **Типографика:** Inter / Manrope / IBM Plex Sans — огромные кегли, 12vw заголовки.
- **Лэйаут:** свободный grid, glassmorphism-карточки, «дышащие» интерактивные градиенты, реакция на курсор.
- **Артефакты:**
  - Three.js — медленно вращающаяся низкополигональная сфера/частицы, реагирующие на курсор.
  - Интерактивный курсор-диск (mix-blend-mode: difference).
  - **AI-консоль** — поле ввода с заранее заданными ответами в духе LLM (без реальной модели).
  - Live-метрики «нейро-погода Рунета» (псевдо-данные).
- **Звук:** мягкий ambient-pad/sub-bass на фоне (опционально, по умолчанию выключен).

## 4. Глитч-переход

- Длительность: 700–1200 мс.
- Слои: RGB-channel split → горизонтальный displacement (SVG `feDisplacementMap` или WebGL-shader) → scanlines → колор-инверсия на 60 мс → resolve.
- Опционально: «сгорающая киноплёнка» — белая вспышка + chromatic aberration в финале.
- При `prefers-reduced-motion: reduce` глитч заменяется на короткий fade (180 мс).

---

## 5. Пасхалки (Easter Eggs)

- **Эра 1:** последовательность ↑↑↓↓←→←→ВА (Konami) → летающие «огоньки» по экрану и `[ЭТО САЙТ ОБНОВЛЁН!]`.
- **Эра 1:** клик по «under construction» 5 раз → запуск «диал-апа».
- **Эра 2:** кнопка «Дуров, верни стену» → стена встряхивается, появляется пост от Дурова.
- **Эра 3:** ввод `/glitch` в AI-консоль → принудительный глитч-переход.
- **Глобально:** в подвале — крошечная пиксельная иконка дискеты, по клику — модалка с «changelog Рунета».

## 6. Технологии

- **Build:** Vite + React 18 + TypeScript.
- **Стили:** Tailwind CSS + CSS-модули для эпох (изоляция стилей по эпохе через `data-era`).
- **Анимация:** GSAP (timeline для глитча и переходов), Framer Motion допускается для микро-UI.
- **3D:** Three.js (через `@react-three/fiber` + `drei`) — только в эпохе 3, lazy-loaded.
- **Шрифты:** локальные/`@fontsource` — VT323/Press Start (эра 1), Tahoma/Verdana/system (эра 2), Inter/Manrope (эра 3). Fallback на `serif`/`sans-serif`.
- **Звуки:** Web Audio API, маленькие сэмплы (≤ 50 KB), отключены по умолчанию.
- **Состояние:** React Context + URL-параметр `?era=1|2|3` (deep-link, F5 сохраняет эпоху).

---

## 7. Доступность и UX

- Поддержка `prefers-reduced-motion`.
- Все интерактивные элементы — клавиатурно достижимы, фокус-стили видны.
- Контраст AA (WCAG) — особенно в эпохе 1 (учесть кислотные цвета).
- Глобальный mute-тоггл, доступен с любой эпохи.
- Адаптив: desktop (≥ 1024), tablet (≥ 768), mobile (≥ 360). На мобиле артефакты упрощаются, Three.js — в облегчённом режиме.

---

## 8. Перформанс

- LCP < 2.5 c, CLS < 0.1.
- Three.js и тяжёлые ассеты эпохи 3 — `React.lazy` + Suspense.
- Изображения — SVG/WebP, гифки эпохи 1 — заменяются на CSS-анимации (никаких реальных гиф ≥ 200 KB).
- Lighthouse Performance ≥ 80 на desktop.

## 9. SEO и метаданные

- `<title>`: «Эхо Рунета — иммерсивное путешествие по эпохам Рунета».
- `<meta description>`: одно предложение-манифест.

## 11. Критерии приёмки

1. Билд проходит (`npm run build`) без ошибок и предупреждений типов.
2. Три эпохи визуально, типографически и по поведению **радикально отличаются** друг от друга.
3. Глитч-переход срабатывает между любыми двумя эпохами и читается как «разрыв реальности».
4. В каждой эпохе есть минимум 3 интерактивных артефакта и 1 пасхалка.
5. Работает deep-link `?era=N` и кнопка «назад» в браузере.
6. Соблюдено `prefers-reduced-motion`.
7. Сайт деплоится и доступен по публичному URL.
8. На главной `<title>` и OG-метаданные корректные.

## 12. Out of scope (на будущее)
