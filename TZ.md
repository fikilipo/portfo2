# ТЗ: «Эхо Рунета» — иммерсивный веб-перформанс

> Версия 2.0 — расширенная и структурированная редакция исходного брифа.
> Цель — сохранить креативный дух оригинала, но задать чёткие границы, по которым проект можно реализовать «под ключ» и оценить.

---

## 1. Концепция (что именно мы делаем)

«Эхо Рунета» — это не сайт-портфолио и не блог. Это **цифровое исследование души российского интернета**, оформленное как одностраничный иммерсивный перформанс.

Пользователь проходит три эпохи Рунета. Каждая эпоха — это отдельный «слой реальности», у которого:

- своя визуальная эстетика;
- своя типографика и палитра;
- своё поведение интерфейса (курсор, ссылки, наведение, скролл);
- свои интерактивные артефакты (счётчики, кнопки, окна, чаты);
- свои пасхалки.

Между эпохами — **глитч-переход** («разрыв реальности»): RGB-сдвиг, деформация, имитация горящей плёнки.

Пользователь должен почувствовать «запах интернета 90-х и холодный блеск будущего».

---

## 2. Эпохи (контент-каркас)

### Эпоха 1. «Дикий Запад» (1997–2003)
**Дух:** Народ.ру, первые форумы, чаты типа «Кроватка», искренность, плохая (но честная) вёрстка.

**Эстетика:**
- табличная вёрстка, плиточный фон (звёзды / огоньки / клетка);
- палитра: насыщенный чёрный, лайм `#39ff14`, маджента, кислотные синий/жёлтый;
- шрифты: Times New Roman / Comic Sans MS / системный моноширинный;
- кастомный пиксельный курсор + «комета» из звёздочек за курсором;
- `<marquee>`-подобные бегущие строки и `blink`-эффект на ссылках.

**Артефакты времени:**
- счётчик посещений в стиле LED-табло;
- баннер «Сайт находится в разработке» (анимированная стройка);
- набор web-баннеров 88×31 (Народ.ру, Rambler ТОП100, NBSP);
- «гостевая книга» (форма с комическими записями);
- блок «Мои друзья» — кружок аватарок-кнопок;
- фоновая midi-кнопка («включить музыку» — опционально, по клику).

**Тексты:** ламповые, с восклицательными знаками, эмотиконами `:-)`, словами вроде «Привет, странник!».

---

### Эпоха 2. «Золотой век / Дуров, верни стену» (2007–2012)
**Дух:** Web 2.0, ранний ВКонтакте, Хабр, демотиваторы, скевоморфизм.

**Эстетика:**
- глянцевые кнопки с градиентом и рефлексом;
- мягкие тени, скруглённые углы (8–12px);
- палитра: бело-голубая (`#e7eef7` → `#5181b8`), акценты — оранжевый `#ff8c00`;
- шрифты: Verdana / Tahoma / системный sans;
- курсор — стандартный, но с лёгким «glow» при наведении на интерактив.

**Артефакты времени:**
- мок-блок «стены ВКонтакте»: пост, лайк, репост, комментарии (счётчики анимируются);
- демотиватор-генератор (чёрная рамка + двухстрочная подпись, можно ввести свой текст);
- блок «друзья онлайн» (зелёные точки);
- кнопка-ссылка «нравится» с +1 и пульсацией сердечка;
- эмблема «Хабр»-стайл цитата с плюсами.

**Тексты:** чуть более сдержанные, мемные («Это норма», «Превед, медвед», «Минусы будут?»).

---

### Эпоха 3. «Экосистема / Нейро-Синтез» (2020–2026)
**Дух:** современный российский бигтех (Яндекс, Сбер, VK, Т-Банк), AI-ассистенты, минимализм.

**Эстетика:**
- тёмная тема, чёрно-графитовый фон с подвижным градиентным mesh;
- крупная типографика (`clamp(48px, 8vw, 160px)`), Inter / Manrope / системный sans;
- glassmorphism-карточки с blur и тонкой границей;
- анимированный SVG-логотип-«нейрон»;
- интерактивный градиент, реагирующий на курсор;
- курсор — кастомный «магнитный» dot + ring.

**Артефакты времени:**
- AI-ассистент в правом нижнем углу с печатающимся ответом;
- карточка «Нейроитог эпохи» — короткое саммари в стиле LLM;
- секция «датасет» с метриками (анимированные числа: домены, мемы, MAU);
- горячая клавиша `/` открывает командную палитру.

**Тексты:** короткие, уверенные, с лёгким холодком («Мы научили интернет понимать тебя»).

## 3. Структура страницы

Single-page, секции рендерятся последовательно (без роутера):

1. **Intro / Boot** — короткая загрузка-«инициализация», CRT-разогрев экрана, выбор «войти».
2. **Era 1 — Дикий Запад.**
3. **Glitch-переход 1 → 2.**
4. **Era 2 — Золотой век.**
5. **Glitch-переход 2 → 3.**
6. **Era 3 — Экосистема.**
7. **Финал / Outro** — таймлайн всех трёх эпох + подпись автора.

Навигация:
- сверху — фиксированный HUD с тремя «вкладками эпох» и индикатором текущей;
- стрелки `←`/`→` и колесо мыши переключают эпохи;
- HUD сам подстраивается под визуал текущей эпохи (меняет шрифт/цвет).

---

## 4. Глитч-переход (отдельный компонент)

Не «просто шум». Полноэкранный overlay длительностью ~900–1400 мс:

1. RGB-split текущего DOM-снимка (canvas).
2. Горизонтальные «разрывы» (slice-glitch) со смещением.
3. Имитация сгорающей плёнки — белая вспышка + чёрные пятна по краям.
4. Короткий звук помехи (опционально, mute по умолчанию, разблокируется первым кликом).
5. После — «вспышка света» и появление новой эпохи.

Реализация: GSAP timeline + canvas snapshot (html2canvas или ручной CSS-overlay со SVG-фильтрами `feTurbulence` + `feDisplacementMap`).

---

## 5. Пасхалки (Easter Eggs)

- **Эра 1:** Konami-код (`↑↑↓↓←→←→BA`) → на странице «загорается» ASCII-огонь и появляется секретная гиф-кнопка «спасибо за посещение».
- **Эра 2:** двойной клик по сердечку «нравится» → каскад из ~36 пиксельных сердечек с физикой (взрыв с разлётом и затуханием за ~2 сек).
- **Эра 3:** `/` → командная палитра с командами быстрого перехода между эпохами, `summon-ai` и `about`. Команда `summon-ai` запускает «ответ нейросети» о смысле эпохи (печатается с typewriter-эффектом).

## 6. Технические требования

**Стек (фиксируем):**
- Vite + React 18 + TypeScript;
- Tailwind CSS (с тремя «темами» эпох через CSS-переменные);
- GSAP (анимации и timeline для глитча);
- Three.js — **опционально**, только если без него никак (можно ограничиться SVG/CSS/canvas);
- ESLint + Prettier.

**Совместимость:**
- desktop-first, 1280px+ — основной сценарий;
- адаптив до 768px (mobile): эпохи переключаются свайпом, тяжёлые эффекты упрощены;
- редьюс-моушн (`prefers-reduced-motion`): глитч и параллаксы выключаются.

**Производительность:**
- Lighthouse Performance ≥ 70 на desktop;
- ленивая инициализация эффектов эпохи только когда она активна;
- глитч-переход — не более ~1.5 сек.

**Доступность (минимум):**
- семантические `<section>` / `<nav>` / `<button>`;
- alt-тексты для значимых изображений;
- focus-visible на всех интерактивах;
- крупные шрифты в Эре 3, контраст ≥ 4.5 (кроме декоративных элементов Эры 1, где «грязная» вёрстка — часть концепта).

**Деплой:**
- Vercel/Netlify-friendly (`npm run build` → `dist/`);
- README с инструкцией: `npm i`, `npm run dev`, `npm run build`.

## 7. Что считаем «готово» (Definition of Done)

- [ ] Проект собирается `npm run build` без ошибок и предупреждений.
- [ ] Все три эпохи реализованы и визуально различимы с первого взгляда.
- [ ] Между эпохами работает глитч-переход.
- [ ] Реализована хотя бы одна пасхалка на эпоху.
- [ ] Сайт работает на 1920×1080 и 390×844 (iPhone-ширина).
- [ ] Есть intro и outro.
- [ ] README объясняет, как запустить и как «прочитать» сайт.
- [ ] Код разбит на компоненты, нет «мегафайлов» по 1000 строк.

## 8. Что вне скоупа (на будущие итерации)

- Бэкенд / админка / CMS.
- Многоязычность (UI только на русском).
- Реальная авторизация / база данных.
- Полноценный звуковой саундтрек по эпохам (только короткие SFX по клику).
